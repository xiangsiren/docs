1. reids 数据类型：

   字符串（string）哈希（hash）列表（list）集合（sets）有序集合（sorted sets) 等类型

   

2. redis单线程快的原因：

   1. 大部分都是纯粹的内存操作
   2. 采用单线程，避免了不必要的上下文切换和竞态条件
   3. 采用单线程i/o多路复用机制

3. redis好处：

   速度快，支持丰富数据类型，支持事务，丰富的特性（缓存，消息，过期自动删除）

4. Redis 相比 memcache的优势：

   可以持久化数据，支持数据的备份（既master-slave 模式的数据备份），丰富的数据类型
